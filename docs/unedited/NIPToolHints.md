\<font class=\"title\"\>+NIP TOOL: Hints\</font\>

\<font class=\"query\"\>SYNTAX:\</font\>

`+nip "hints"`

\<font class=\"query\"\>AFFECTED / CONCERNED LIBS/ETC.:\</font\>

\* [Emoting](NIPLibRefEmoting) (hints) \* [How-to: hints](NIPHowHints)

\<font class=\"query\"\>GENERAL DESCRIPTION:\</font\>

\<img src=\"%ATTACHURLPATH%/hints.png\" alt=\"hints.png\" /\>

The \"hints\" +nip tool is used to administer the rumors/hints data in
NPC\'s. The tool functions in two ways. One is to start it from a
regular body. The tool will then ask for the woename of the hints data
object you wish to modify. The other is to possess the NPC whose hints
data you wish to modify, and then executing the command. The command
will then presume that you wish to edit that NPC\'s hints data,
presuming the NPC has hints data recorded.

\-- Main.KalleAlm - 12 May 2005
